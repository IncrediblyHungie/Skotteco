# Portfolio Demo Sites Design

## Goal
Replace the 2 similar-looking portfolio demo sites with 4 visually distinct, high-quality single-page sites that showcase Skotte's range.

## Sites

### 1. Sunrise Cafe (cafe-after.html)
- **Vibe**: Warm, cozy, artisan
- **Palette**: Cream (#FDFBF7), warm brown (#2C2416), terracotta (#C4853A)
- **Layout**: Editorial — big hero, menu grid, warm textures

### 2. Pacific Dental (dentist-after.html)
- **Vibe**: Clean, trustworthy, modern
- **Palette**: White (#FAFBFD), navy (#0F172A), teal (#2563EB)
- **Layout**: Clinical-modern — trust badges, team cards, appointment CTA

### 3. Velvet Hair Studio (salon-after.html)
- **Vibe**: Chic, editorial, feminine
- **Palette**: Black (#111), blush pink (#E8B4B8), gold (#C9A96E)
- **Layout**: Fashion-forward — full-bleed imagery, minimal text, elegant type

### 4. Iron Athletics (gym-after.html)
- **Vibe**: Bold, energetic, raw
- **Palette**: Dark (#0A0A0A), neon green (#39FF14) or orange, concrete gray
- **Layout**: High-contrast — big bold type, dark bg, aggressive CTAs

## Structure (each site)
- Self-contained HTML with inline CSS (no external deps)
- Fixed header with logo + nav
- Hero section with headline + CTA
- 2-3 content sections
- Footer
- System fonts only

## Homepage Changes
- Add salon + gym work-cards to index.html
- Portfolio grid: 2x2 layout

## Approach
4 parallel subagents, each building one site independently from a creative brief.
